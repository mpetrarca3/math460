# Plot the Testing R-Squared Accuracy of adaboosts with n_estimators running from n1 to n2 and two learning rates l1 and l2. 
np.random.seed(64)
n1 = 1
n2 = 100
l1 = .1
l2 = 1

ac = pd.DataFrame([], columns=list(['Rounds','Learning Rate','Testing Rsquared']))
from sklearn.tree import DecisionTreeClassifier
for rs in range(n1, n2):
    for lr in [l1, l2]:
        boost = AdaBoostRegressor(n_estimators=rs, learning_rate=lr)
        boost.fit(x_train,y_train)
        ac = ac.append(pd.DataFrame([[rs, lr, boost.score(x_test,y_test) ]], 
                                    columns=list(['Rounds','Learning Rate','Testing Rsquared'])), ignore_index=True)

import seaborn as sns; sns.set()
import matplotlib.pyplot as plt
ax = sns.lineplot(x="Rounds", y="Testing Rsquared", hue =ac['Learning Rate'].astype('category'),data=ac)
ax.text(x=0.5, y=1.1, s='Learning fast vs. slow', fontsize=16, weight='bold', ha='center', va='bottom', transform=ax.transAxes)
Out[15]:
Text(0.5, 1.1, 'Learning fast vs. slow')

The final model produced a testing R-Squared value of ~0.4723.

Final Parameters:

n_estimators = 2, learning_rate = 1

The learning rate was set to 1 since it outperformed the learning rate of 0.1 on the graph above at low values of n_estimators. As the value of n_estimators increases, the testing R-Squared decreases. Therefore n_estimators was set to 2.
